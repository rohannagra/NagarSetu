**Issue:** MongoDB not set up yet
**Impact:** Backend shows errors when submitting complaints
**Solution:** Set up MongoDB Atlas (5 minutes)

**Options:**
1. **MongoDB Atlas (Recommended):** Cloud, free, no installation
2. **Local MongoDB:** Install on your machine

**Steps for Atlas:**
1. Sign up: https://www.mongodb.com/cloud/atlas/register
2. Create free cluster
3. Get connection string
4. Update `server/.env`
5. Restart backend

**Detailed guide:** See `docs/MONGODB_SETUP.md`
