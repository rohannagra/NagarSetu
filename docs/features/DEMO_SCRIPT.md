## 🎯 Demo Flow (5 Minutes)

### Introduction (30 seconds)

**Show:** Landing page
- Beautiful hero section
- Animated statistics
- Feature showcase

---

### Part 1: Citizen Journey (2 minutes)

#### A. Register & Login (20 seconds)
1. Click "Get Started"
2. Quick registration
3. Login with credentials
4. **Show:** Dashboard with welcome message

#### B. Submit Complaint (60 seconds)
**Say:** "Citizens can submit complaints through our intelligent 3-step form."

**Step 1 - Details:**
- Enter title: "Broken streetlight on MG Road"
- Select category: Street Lights (show icon selection)
- Add description
- **Show:** File upload with preview
- Click Next

**Step 2 - Location:**
- Click "Use Current Location" OR
- **Show:** Interactive map - click to select location
- Enter address details
- Click Next

**Step 3 - Review:**
- **Show:** Complete summary
- Click "Submit Complaint"
- **Show:** Success message

#### C. View Complaints (40 seconds)
- Navigate to "My Complaints"
- **Show:** Search bar
- **Show:** Filters (status, category, priority)
- **Show:** Beautiful complaint cards with badges
- Click any complaint

**Show complaint detail page:**
- Tabs: Details, Timeline, Chat
- **AI Analysis section:**
  - Predicted category
  - Sentiment analysis
  - Urgency score (show progress bar)
  - Keywords
- Interactive map with location
- Status timeline
- Media attachments (if uploaded)

     - Resolved, In Progress, Pending, Closed
   - **Progress Bars:** System health metrics
     - Resolution rate
     - Officer utilization
     - Citizen satisfaction
     - SLA compliance

3. **Quick actions:**
   - Manage users
   - Manage departments
   - View analytics
   - All complaints
